FastAPI Campaign CRUD

A learning project built with FastAPI, Python, and SQLModel to implement basic CRUD operations on a Campaign entity using an SQL ORM.

Tech Stack

Python

FastAPI

SQLModel (SQLAlchemy + Pydantic)

SQLite

Uvicorn

Features

Create, Read, Update, Delete Campaigns

ORM-based database handling
Purpose

This project is meant to practice FastAPI, SQL ORM concepts, and CRUD API design.
Not intended for production use.

Dependency-injected DB sessions

Auto timestamp using UTC

Campaign Fields

campaign_id (Primary Key)

name

due_date

created_at
